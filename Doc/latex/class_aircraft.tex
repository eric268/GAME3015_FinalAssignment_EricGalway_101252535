\doxysubsection{Detailed Description}
Concrete class for \mbox{\hyperlink{class_aircraft}{Aircraft}}. 

This is the concrete class for all aircraft. The current available aircraft options are the Eagle or Raptor. This class inherits from the \mbox{\hyperlink{class_entity}{Entity}} class.
